A dashboard plugin for [KOReader](https://github.com/koreader/koreader).

## Concept

A single home screen summarizing what matters across your reading and your installed game plugins: the book you're currently reading, recently played games with session stats, and quick actions — instead of digging through separate menus.

## Features

- **Reading section** — currently-open book and recent reading history
- **Recent games** — installed vs. played plugin counts, per-game session stats (sessions, time played)
- **Quick actions** — update plugins, jump to the library
- **Startup display** — optionally show the dashboard automatically when KOReader launches, with a configurable delay
- **Home-button shortcut** — optionally bind the device home button to open the dashboard

## Controls

Accessed from the menu → **Tools** → **Dashboard**, or automatically at startup if enabled.

| Setting | Description |
|---------|-------------|
| Show at startup | Toggle automatic display on launch |
| Startup delay | 0.5 / 1 / 2 seconds before the dashboard appears |
| Home button → Dashboard | Bind the device home button to open the dashboard |

## Installation

1. Download `dashboard.koplugin.zip` from the [latest release](../../releases/latest).
2. Extract into the `plugins/` folder of your KOReader data directory.
3. Restart KOReader.
